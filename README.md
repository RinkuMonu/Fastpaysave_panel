# FASTag Admin Panel (React + Vite + Tailwind)

Simple starter admin panel with:

- Dashboard (stats + quick actions)
- User Management (list + create/edit form UI)
- FASTag Recharge screen (form + recent transactions table)
- Settings placeholder (for API configs)

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- React Icons

## Getting Started

```bash
npm install
npm run dev
```

Open the URL shown in terminal (default: http://localhost:5173).

Then integrate your actual APIs inside:

- `src/pages/Users.jsx`
- `src/pages/FastagRecharge.jsx`
- `src/pages/Settings.jsx`
